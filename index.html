Data We Collect

Cryzer Bot collects and stores the following information to provide its services:

User Data

  • Discord User ID - To identify users across servers
  • Username & Display Name - For display purposes
  • XP and Level Data - For the leveling system
  • Premium Status - To track premium subscriptions
  • Warnings and Moderation History - For moderation purposes
  • Custom Configurations - User preferences and settings

Server (Guild) Data

  • Discord Server (Guild) ID - To identify servers
  • Server Name - For display purposes
  • Channel IDs - For configured features (logs, announcements, music, etc.)
  • Role IDs - For auto-roles and permission management
  • Custom Prefixes - Server-specific command prefixes
  • Feature Configurations - Settings for leveling, moderation, tickets, etc.
  • Ticket Data - Ticket transcripts and configurations
  • Giveaway Data - Active and past giveaway information
  • YouTube Notification Subscriptions - For YouTube update notifications

Temporary Data

  • Voice Channel Sessions - For Join-to-Create voice features
  • Music Queue Data - Currently playing songs (cleared on disconnect)
  • Command Usage Logs - For analytics and debugging (retained temporarily)

How We Use Your Data

We use the collected data to:

  • Provide core bot functionality (music, moderation, leveling, etc.)
  • Save user preferences and server configurations
  • Track user progression and achievements
  • Manage tickets, giveaways, and events
  • Send notifications for configured services
  • Improve bot performance and fix issues
  • Prevent abuse and enforce terms of service

Data Storage and Security

  • Data is stored in a MongoDB database with industry-standard security measures
  • We implement encryption and access controls to protect your data
  • Only authorized developers have access to the database
  • We do not sell or share your data with third parties
  • Data is retained only as long as necessary to provide bot services

Your Rights

You have the right to:

  • Access your data - Request a copy of data we store about you
  • Delete your data - Request deletion of your personal information
  • Correct your data - Request corrections to inaccurate data
  • Opt-out - Stop using the bot at any time by removing it from your server

Data Deletion Requests

To request deletion of your data, you can:

  1. Use the /privacy command in any server with the bot
  2. Contact us directly via our support server: [Your Support Server Link]
  3. Email us at: [Your Contact Email]

We will process deletion requests within 30 days. Please note:

  • Deleting user data will reset your XP, levels, and all personal stats
  • Server owners can request deletion of all server-related data
  • Some data may be retained for legal or security purposes (e.g., ban records)
